### 啥功能啊

* 天天基金收益榜十一览
* 自持仓位基金一览
* 三大指数一览
* 添加自选基金
* 删除自选基金
* 拖拽排序
* 手动/自动刷新
* 最后你以为我是闲的？不，因为目前还没发现哪个mac端软件有此类功能

### 使用说明书

为了个人财产隐私保护，本luer设计了两套数据源模式，一套显示自己关注的或别人推荐的基金，一套显示自己已经持仓的基金，并且可以根据持仓金额，
持仓金额，持仓金额，不是份额，依据实时估值计算出当日大概收益。咋切换呢？很明显，点页面上的（推荐源\自选源）按钮!
