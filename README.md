### 都有啥功能啊

* 天天基金月收益榜三十全览
* 待考察基金一览
* 已买入基金一览
* 三大指数一览
* 添加自选基金
* 删除自选基金
* 拖拽排序
* 计算上日持仓净值
* 手动/自动刷新
* 最后你以为我是闲的？不，因为目前还没发现哪个Mac端软件有此类功能

**2. 非普通安装**

### 使用说明书

本luer设计了三套数据源模式：
#### 榜单区：数据源自天天基金月收益榜单前三十位，供大家分析整体基市趋势行情
#### 观察区：想买入但是未买入的基金，可以放在这里相机而行
#### 持有区：自己已经买入的基金，可输入持仓金额（是钱不是份额）查看实时预估收益
* 注：三套数据源相互隔离，互不影响
